Project Overview: Pastry Portfolio for Chef Jazmin Lester

Application Description

This project is a professional portfolio website for Pastry Chef Jazmin Lester, focused on showcasing her custom cakes, desserts, and bakery collaborations. The application will centralize her visual work, core menu offerings, and current availability, while providing a clear and structured way for customers to submit non-transactional order inquiries.

The website’s purpose is to increase Jazmin’s online visibility, present her work in a clean and modern layout, and streamline how potential clients learn about her services, flavor options, and schedule.

Intended Users

  • Local customers seeking custom cakes and dessert orders for events.
  • Event planners organizing weddings, birthdays, showers, and corporate events.
  • Bakery owners or managers interested in collaborations.
  • Friends, followers, and returning customers who want to browse her portfolio.

Overview of Website Content

  • Chef bio and story background.
  • Curated gallery of cakes, cupcakes, and specialty desserts.
  • Menu with flavor options and starting price ranges.
  • Order inquiry form for custom requests (front-end only).
  • Weekly schedule and bakery location details.
  • Optional testimonials and FAQs.

Page Design Details

Each page will follow the site’s overall layout and style, maintaining a consistent navigation bar, responsive content sections, and dynamic interactivity. Detailed page-by-page breakdowns include:

  • Home: Hero banner, intro text, and featured desserts.
  • About: Chef background, photos, and philosophy.
  • Menu & Gallery: Category tabs, image filters, and lightbox functionality.
  • Order Inquiry: Validated form with dropdowns and date picker.
  • Schedule & Location: AJAX-loaded schedule with embedded map.
  • Testimonials / FAQ: Accordion layout using jQuery UI (optional).

Planned Dynamic Functionality

  • Interactive Gallery: Filterable desserts with modal image viewer.
  • jQuery UI Tabs/Accordion: For menu categories or FAQ toggles.
  • Validated Form: Client-side form checking on Order page.
  • AJAX Schedule Loader: Loads weekly hours from JSON for Schedule page.
